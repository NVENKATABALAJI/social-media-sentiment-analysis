# 🚀 Project Overview

Sentilytics is an AI-driven Social Media Sentiment Analysis platform that collects YouTube comments and transforms them into meaningful business insights.

The application helps organizations, content creators, marketing teams, and researchers understand audience perception by analyzing sentiment trends, emotions, and engagement patterns.

The platform is designed to provide an industry-level dashboard experience rather than a traditional academic project interface.

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/NVENKATABALAJI/social-media-sentiment-analysis.git
```

## 2️⃣ Navigate to Project

```bash
cd social-media-sentiment-analysis
```

## 3️⃣ Create Virtual Environment

```bash
python -m venv .venv
```

## 4️⃣ Activate Virtual Environment

Windows:

```bash
.venv\Scripts\activate
```

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 6️⃣ Run Application

```bash
streamlit run app.py
```
